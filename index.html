Google Sheets 연결
Apps Script 웹 앱 URL을 입력하세요.
예수금 설정
설정 가이드
Step 1. Google Sheets에서 확장 프로그램 → Apps Script 열기
Step 2. AppScript_Code.gs 파일 내용 전체 붙여넣기
Step 3. setupHeaders() 함수 실행 → 초기 데이터 생성
Step 4. 배포 → 웹 앱 → 액세스: 모든 사용자 → 배포
Step 5. 생성된 URL을 위 입력란에 붙여넣고 저장
